php_evaluator for Drupal 6
=============

AJAX PHPcode evaluator for drupal site. Evaluate PHP code and return results via AJAX.  
Can be used for drupal sites development process.

INSTALLATION
Install and enable as usualy.

USING
After the module has been enabled, it's provide a textarea with 
syntax hightlight which can be found here /admin/evaluate.

DEPENDECIES
Module using codemirror library to hightlight syntax.
